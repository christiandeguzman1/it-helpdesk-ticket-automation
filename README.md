## Internal IT Helpdesk Ticket Automation System

## Overview
A web-based internal IT helpdesk system built with Flask and SQLite.

This system allows employees to:
- Submit IT support tickets
- Automatically categorize issues using keyword detection
- Automatically assign priority levels
- Track ticket status updates

## Features
- Automatic issue categorization
- Automatic priority assignment
- Admin dashboard for ticket management
- Status updates (Open, In Progress, Closed)
- SQLite database storage

## Tech Stack
- Python
- Flask
- SQLite
- HTML/CSS

## How to Run

1. Clone the repository
2. Create virtual environment:
   python3 -m venv venv
3. Activate environment:
   source venv/bin/activate
4. Install dependencies:
   pip install -r requirements.txt
5. Run application:
   python app.py

Visit:
http://127.0.0.1:5000
